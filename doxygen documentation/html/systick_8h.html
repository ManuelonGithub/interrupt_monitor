typedef struct systick_countdown_ systick_countdown_t

SysTick countdown structure.

SysTick uses this structure to count down the value every tick, once the value reaches zero, it'll stop the count down and call the callback function.

◆ systick_counter_t

typedef struct systick_counter_ systick_counter_t

SysTick counter structure.

SysTick uses this structure to count ticks, if comparision is enabled, it'll check if the count has reached the cmp value, and if it has it'll reset the count and call the callback funtion.
