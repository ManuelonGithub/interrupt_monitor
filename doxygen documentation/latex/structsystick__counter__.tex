\doxysubsection{Detailed Description}
Sys\+Tick counter structure. 

Sys\+Tick uses this structure to count ticks, if comparision is enabled, it\textquotesingle{}ll check if the count has reached the cmp value, and if it has it\textquotesingle{}ll reset the count and call the callback funtion.
